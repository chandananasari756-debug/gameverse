## Overview
GameVerse is a **gaming website** designed to showcase featured games, trailers, and provide a contact form for support. The project demonstrates **Bootstrap 5 component usage**, **responsive design**, and **front-end best practices**.

---

## Project Features
- **Home Page:** Hero section, featured games cards, game trailer carousel.  
- **About Page:** Studio mission, team members, and services.  
- **Contact Page:** Responsive contact form with success alert message.  
- **Consistent Navbar & Footer** across all pages.  
- Fully **responsive design** for desktop, tablet, and mobile.  

---

## Technologies Used
- **HTML5** for page structure  
- **CSS3 & Bootstrap 5** for styling and responsive layout  
- **JavaScript** for interactive contact form  
- **Git & GitHub** for version control and deployment  

---

## Challenges & Solutions
- **Carousel images not appearing:** Placed images correctly in `assets/images/` folder.  
- **Contact form submission error:** Solved by using JavaScript to display a success message instead of real form submission.  
- **Responsive spacing adjustments:** Used Bootstrap utility classes for margins, padding, and grid layouts.  

---

## Learning Outcomes
- Learned to **remix Bootstrap 5 components** creatively.  
- Gained hands-on experience with **responsive UI design**.  
- Practiced **version control with GitHub** and deployment using GitHub Pages.  
- Improved **UI/UX design sense** and page composition skills.
